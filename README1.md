# 基于STC89C52RC单片机的多功能系统

这是一个基于STC89C52RC单片机开发的多功能嵌入式系统项目，集成了温度监测、音乐播放、倒计时和信息显示等功能。

## 功能模块

### 1. 温度监测模块
- 使用DS18B20数字温度传感器实时监测环境温度
- 在LCD1602显示屏上显示当前温度值
- 精确到小数点后一位

### 2. 音乐播放模块
- 实现《春日影》（Haruhikage）音乐播放功能
- 通过蜂鸣器播放预设的音乐旋律
- 数码管倒计时3秒后开始播放

### 3. 倒计时模块
- 可设置0-59分钟和0-59秒的倒计时
- 实时时钟显示在LCD上
- 倒计时结束时显示"It's Time!"提示信息
- 蜂鸣器提醒功能

### 4. 信息显示模块
- 显示系统设计者信息
- 显示开发环境信息
- 显示AI助手信息
- LED流水灯效果展示

## 硬件组成

- STC89C52RC单片机主控
- DS18B20温度传感器
- LCD1602液晶显示屏
- 矩阵按键（4x4）
- 蜂鸣器
- 8位LED指示灯
- 数码管显示

## 操作说明

1. **主界面**：系统启动后显示主界面，通过矩阵按键1-4选择功能
2. **温度监测**：按矩阵按键1进入温度显示模式
3. **音乐播放**：按矩阵按键2进入音乐播放模式，再按矩阵按键3开始播放
4. **倒计时**：按矩阵按键3进入倒计时模式
   - 使用按键1/2增加分钟/秒钟
   - 使用按键5/6减少分钟/秒钟
   - 按键3启动/停止倒计时
5. **信息显示**：按矩阵按键4进入信息显示模式

## 开发环境

- 编译器：SDCC (Small Device C Compiler)
- 构建工具：PlatformIO
- 开发板：STC89C52RC
- 晶振频率：11.0592MHz

## 特色功能

1. **模块化设计**：各功能模块独立实现，便于维护和扩展
2. **中断驱动**：使用定时器中断实现精确的倒计时功能
3. **实时显示**：LCD实时显示系统状态和相关信息
4. **交互友好**：通过矩阵按键实现人机交互
5. **视觉效果**：LED流水灯增强用户体验

## 注意事项

1. 系统使用11.0592MHz晶振，确保定时精度
2. 倒计时功能在结束时会显示"It's Time!"并发出蜂鸣提示
3. 音乐播放功能需要3秒倒计时准备时间
4. 所有功能可通过外部按键3返回主界面
